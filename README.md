# homework-12
# Отчёт о форензическом анализе сетевых дампов

## Описание

Скрипт `homwork12.py` выполняет форензический анализ pcap/pcapng-файлов: извлекает артефакты (DNS, HTTP, ICMP, соединения) и строит статистику и графики.

## Тестовые дампы

| Файл | Тип трафика | Пакетов | Объём |
|------|-------------|---------|-------|
| `traffic_raw.pcap` | Смешанный (TCP, HTTP, HTTPS) | 71 088 | ~84 МБ |
| `dhcp.pcapng` | DHCP | 4 | ~1.3 КБ |
